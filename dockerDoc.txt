### **2. Create a Dockerfile**
Inside the repository, create a file named `Dockerfile` with the following content:

```dockerfile
# Use the official Ubuntu slim image

# Set up basic tools

    bash \
    vim \
    curl \
    git \
```


   ```bash
   docker run -it portable-linux-vm
   ```
